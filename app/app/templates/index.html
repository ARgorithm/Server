ARgorithm Server is running

Connect to the server using the Toolkit and start making your ARgorithms.

If running toolkit on same machine as server just use the --local option in the command line arguments. If using toolkit on a different machine then on the command line use: 

$ ARgorithm configure

and then copy the url of this page and paste it there. The same URL should be added as server endpoint in mobile application.
